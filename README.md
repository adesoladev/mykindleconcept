# 🌹 MyKindle Concept Website

A premium, romantic experience website designed for **MyKindle Concept**, a brand focused on creating intimate and unforgettable moments for couples.

This project is a fully responsive single-page website built using **HTML, Tailwind CSS, and Vanilla JavaScript**.

---

## 💡 About the Project

MyKindle Concept is a romantic experience brand that helps couples celebrate love through thoughtfully designed moments such as:

- Romantic date experiences
- Proposal planning
- Anniversary celebrations
- Bridal showers
- Honeymoon planning
- Couples experience events

The website is designed to feel emotional, elegant, and premium while guiding users to book or inquire about experiences.

---

## 🎯 Project Goal

The goal of this website is to:

- Create a strong emotional first impression
- Showcase romantic experiences clearly and beautifully
- Build trust with potential clients
- Drive bookings via contact form or WhatsApp
- Present the brand as premium and intentional

---

## 🧱 Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- FontAwesome (for icons)

---

## 🎨 Design Direction

The design follows a soft luxury aesthetic:

- Romantic and emotional feel
- Clean and minimal layout
- Soft blush, ivory, and champagne gold color palette
- Elegant typography (serif for headings, sans-serif for body)
- Smooth scrolling and subtle animations
- Mobile-first responsive design

---

## 🧭 Website Structure

The website is structured as a single-page layout with the following sections:

- Home (Hero Section)
- About
- Experiences
- Couples Delight (Signature Experience)
- Gallery
- Testimonials
- Contact
- Footer

---

## ✨ Key Features

- Sticky navigation bar with smooth scrolling
- Responsive hamburger menu for mobile
- Emotionally driven hero section
- Service cards for romantic experiences
- Signature Couples Delight section
- Image gallery layout
- Testimonials section
- Contact form with WhatsApp integration
- Floating scroll-to-top button
- Fully responsive design across all devices

---

## 📱 Responsiveness

The website is fully optimized for:

- Mobile devices
- Tablets
- Desktop screens
